# TODO-LIST
Todo List app in rust with Rocket &amp; Diesel

## Install and run project

### Install rust:
```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Set rust nightly version (in project folder):
```shell
rustup override set nightly
```


### Build and run API
```shell
cargo run
```
